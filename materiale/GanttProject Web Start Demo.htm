GanttProject Web Start Demo

What do I need to run this demo?

You need Java 5 Runtime Environment, and you favorite web browser that knows about Web Start. Usually Web Start supports appears in the most popular browser automatically when you install Java5

Learn more: What is WebStart

WebStart is installed on my computer!

Excellent! Just click the button below to start GanttProject demo. Please note that you may be asked to trust our security certificate and grant all access privelegies.

Why do you call it demo? Is there any limited functionality?

Yes, for download time reasons WebStart edition of GanttProject hosted on our site has no HTML/PDF export and Microsoft Project™ import/export functions. As for the rest, it is a full featured GanttProject.
